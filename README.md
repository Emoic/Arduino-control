# Arduino-control
Use arduino mega2560 to control the car

Required modules：

1.Arduino Mega2560

2.Temperature and Humidity (DHT) Sensor （DHT11)

3.Ultrasonic Sensor HC-SR04

4.Infrared Obstacle Avoidance Sensors(x3)

5.LED

6.M3 screws and nuts
